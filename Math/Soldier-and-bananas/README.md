# Soldier and bananas 🍌
Un soldado quiere comprar x bananas, la primera banana tiene un coste n, mientras que la segunda 2n, la siguiente 3n...
El soldado tiene k dolares, y debemos saber cuanto dinero debe pedir prestado (sino necesita mas dinero devolvemos 0)

## Como funciona
Calculamos el precio total de las bananas usando un bucle. Posteriormente calculamos la diferencia, si el dinero no alcanza.

# Ejemplos
1.Input: 1 2 1

Output: 0

2.Input: 1 5 6

Output: 16
