# Anagram Checker (C++)

Este programa verifica si dos cadenas de texto son anagramas, es decir, si contienen los mismos caracteres en distinto orden.

## Cómo funciona
El programa compara los caracteres de ambas cadenas y comprueba si coinciden.

## Ejemplo

Input:
abc
cba

Output:
YES
