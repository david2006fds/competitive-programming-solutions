###Comparation (C++)
Este programa compara dos cadenas de texto sin importar mayusculas o minusculas.

##Cómo funciona
El programa compara los caracteres de ambas cadenas y comprueba si son menores, mayores o iguales.

#Ejemplo
Input: Abc abC

Output: YES

Input: abs Abz

Output: NO
